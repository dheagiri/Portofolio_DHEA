Tutorial: Deploy Website ke GitHub

Langkah 1: Buat Repository di GitHub

Buka GitHub dan login. Klik New di bagian repository atau ikon (+) di pojok kanan atas, lalu pilih New repository. Beri nama repository dan pilih Public. Klik Create repository.

Langkah 2: Tambahkan File ke Repository

Jika file proyek Anda sudah ada di komputer, Anda bisa meng-upload langsung ke GitHub atau menggunakan Git.

  • Opsi A: Upload langsung ke GitHub.
  • Opsi B: Gunakan Git untuk push file dari command line:

            git  init
            git add .
            git commit -m "Initial commit"
            git branch -M main
            git remote add origin https://github.com/username-anda/my-website.git
            git push -u origin main

Langkah 3: Aktifkan GitHub Pages

Buka tab Settings pada repository Anda, gulir ke bagian Pages, pilih branch yang ingin diterbitkan (biasanya main), lalu klik Save.

Langkah 4: Tunggu Hingga Situs Diterbitkan

Tunggu beberapa menit agar situs Anda dapat diakses di URL yang diberikan di pengaturan Pages.
